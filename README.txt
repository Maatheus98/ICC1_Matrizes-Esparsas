Este trabalho é uma implementação de matriz esparsa
através da linguagem C. O programa realiza as funções
de criar matriz, inserir valor a uma posição da matriz, 
consultar valor de uma posição da matriz, exibe a soma
dos valores de uma linha ou coluna e exclusão da matriz.
